## 📌 What is xykt?

xykt is a macOS cutout app based on Apple VisionKit. It helps you remove a subject from a photo with a few clicks. You do not need photo editing skills to use it.

Use it when you want to:
- cut out a person from a photo
- isolate an object from a background
- make a clean image for a post, slide, or document
- save time on basic image editing

## 🧭 How to use xykt

### 1. Open the app
Launch xykt after you install it. The main window should show a simple import or open area.

### 2. Add a photo
Choose the image you want to edit. Most users start with a photo that has a clear person or object in the frame.

### 3. Let the app detect the subject
xykt uses VisionKit to find the main subject in the image. Wait for the cutout to finish.

### 4. Check the result
Review the preview. Look at the edge around the subject and make sure the cutout looks right.

### 5. Save the cutout
Export the result in the format you need. You can keep the background removed or place the subject on a plain background.

## 📷 Best photo types

xykt works best with photos that have:
- one clear subject
- good lighting
- a simple background
- strong contrast between subject and background

It may struggle with:
- crowded scenes
- very dark images
- subjects that blend into the background
- photos with motion blur

If the first result is not clean, try another photo with better light and less clutter.

## 📁 File formats

xykt should work well with common image formats used on macOS, such as:
- JPG
- PNG
- HEIC

For the best export result, use PNG if you want to keep transparency after removing the background.

## 🔍 Tips for better results

- Use a photo with a single main subject
- Keep the subject far from the background when possible
- Use a clear, bright image
- Crop out extra space before you run the cutout
- Save the final image as PNG if you need transparency

These small steps can improve edge quality and make the cutout look cleaner.
